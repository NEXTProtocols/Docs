In this page you will learn how to use the Mocking Data API.

# How to use the Mocking Data API

You only need to change the domaine name in the url : 

- **Production** -> https://api.nextprotocol.fr
- **Development** : https://test.api.nextprotocol.fr


You will then get a random output following the schema of the data you want to mock.

# How see if the Mocking API is working

We can have 3 cases :

<Tabs>
<Tab title="Mocking data works">
The mocking data works and can be used with the test url. You will see the following **Badge** callout.

<Badge icon="database" stroke color="green">Mock data works</Badge>

</Tab>
<Tab title="Mockint data does not work">
The mocking data does not work, but in the future it will be possible to mock the data. You will see the following **Badge** callout.

<Badge icon="database" stroke color="red">Mock data does not work</Badge>
</Tab>
<Tab title="This API cannot be mocked">
The API is not running. You will see the following **Badge** callout.

<Badge icon="database" stroke color="orange">This API cannot be mocked</Badge>
</Tab>

</Tabs>
