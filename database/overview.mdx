## Projets Supabase

Cette section documente les 3 projets Supabase utilisés par NEXTProtocol, incluant :

- **Schémas** : Structure des tables et relations
- **Triggers** : Actions automatiques sur les modifications
- **Functions** : Fonctions PostgreSQL personnalisées
- **Listeners** : Services qui réagissent aux changements
